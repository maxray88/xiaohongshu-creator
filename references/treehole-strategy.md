## 一、账号定位

### 1.1 人设
- **昵称**：情绪树洞 / 深夜树洞 / 心灵解药 / 清醒研究所 / 反内耗基地
- **头像**：深色渐变背景 + 树洞/月亮元素 + 柔光效果，暖粉色调
- **简介**：说说话，被听见 🌙 | 职场反内耗 | 恋爱清醒脑 | 社交焦虑自救 | 当代年轻人精神状态实录
- **签名**：这里没有说教，只有共鸣

### 1.2 内容调性
- **语气**：温柔但不软弱，清醒但不冷漠，像朋友深夜聊天
- **风格**：金句图文为主，沉浸式解压/树洞为辅
- **价值**：每篇笔记让读者感到「被理解」「被治愈」「想转发」

## 三、封面设计规范

### 3.1 视觉风格
- **背景**：高质量情绪类图片（星空、海洋、森林、城市夜景、柔光）
- **主色调**：深蓝/紫色/粉色渐变（治愈感）
- **文字**：白色/金色大字 + 描边 + 发光效果
- **风格**：极简、高级、有氛围感

### 3.2 封面模板（金句图文）
```
[顶部] 系列标签（小字）
[中部] 金句（超大字，88-130px）
[底部] 账号名 + CTA
```

### 3.3 封面模板（沉浸式树洞）
```
[顶部] 树洞emoji + 标题
[中部] 场景描述（中等字号）
[底部] 治愈金句（大字）
```

## 四、发布时间策略

### 4.1 最佳时间
- **工作日**：12:00-13:00（午休）/ 21:00-23:00（深夜情绪高峰）
- **周末**：10:00-12:00 / 20:00-22:00
- **重点**：晚上9点后是情绪类内容流量高峰

### 4.2 发布频率
- **第1-2周**：日更1篇（建立账号权重）
- **第3-4周**：每周5篇（稳定更新）
- **第2个月起**：每周3-4篇（质量优先）

## 五、互动策略

### 5.1 评论区运营
- **置顶评论**：补充更多金句或提问
- **回复策略**：每条回复都带温暖/清醒的观点
- **引导互动**：「你遇到过吗？」「评论区说说你的故事」

### 5.2 话题标签策略
- **固定标签**：#情绪树洞 #心灵治愈 #反内耗 #清醒恋爱 #社恐自救
- **流量标签**：#当代年轻人 #精神状态 #情感共鸣 #治愈系
- **互动标签**：#你同意吗 #评论区聊聊 #扎心了

## 六、数据监控

### 6.1 关键指标
- **曝光量**：目标单篇1000+
- **互动率**：点赞率>5%，评论率>2%（情绪类内容互动率更高）
- **涨粉率**：每1000曝光涨粉8-15人（情绪类涨粉更快）

### 6.2 优化方向
- 曝光低→优化标题和封面
- 互动低→优化内容和CTA
- 涨粉低→增加系列感和人设
