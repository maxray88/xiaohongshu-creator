# card-xiaohongshu Template Spec (Quick Reference)

Source: `/Users/maochundong/open-design/skills/card-xiaohongshu/SKILL.md`

- Size: 1080×1440 (3:4 portrait).
- Count: 3–18 cards (platform limit); 6–9 cards is the practical sweet spot for image-text notes.
- Palette: low-saturation / matte / pink family.
- Layout: wide letter-spacing headline, plenty of whitespace, rounded cards.
- Structure: cover → body cards (one idea per card) → summary → CTA.
- Watermark: author name + date at bottom-right of each card.

Note: Open Design CLI (`od convert`) is currently build-blocked by Node 22 vs ~24. Use this spec as design guidance while HyperFrames remains the active renderer. See `references/opendesign-repo-status.md`.
