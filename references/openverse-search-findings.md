# Openverse Search Results for Anime/Character Queries

## Finding: Character-Specific Searches Return Limited Results

When searching Openverse for specific anime characters like "crayon shinchan":
- `crayon shinchan` → 9 results, but mostly train wraps (Tobu 51058), station photos, and merchandise — NOT character art
- `shinchan cartoon` → 0 results
- `crayon shinchan family` → 0 results
- `shinchan misae mom` → 0 results
- `crayon shinchan anime` → 0 results

**Why**: Openverse aggregates from Flickr, Wikimedia, etc. Character-specific anime art is often copyrighted and not CC-licensed, so it doesn't appear in Openverse results.

## Recommended Approach for Character Posts

1. **Use Openverse for generic/anime-style images**: Search "anime illustration japanese", "kawaii illustration pink", "cute cartoon family"
2. **Compose covers with Pillow**: Add character name as text overlay on generic anime-style backgrounds
3. **Alternative**: Use the character's name in Japanese (e.g., "野原しんちゃん") for better results on some platforms
